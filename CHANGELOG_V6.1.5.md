# V6.1.5｜同步透明度與資料新鮮度

## 公開介面

- 頁首同步狀態改為可點擊按鈕，開啟官方來源同步明細。
- 顯示總統府、立法院、內政部、中選會候選人與中選會歷屆資料筆數。
- 地區資訊卡新增「資料覆蓋」標章，區分完整、部分與等待同步。
- 線上版每 90 秒檢查一次 `data/manifest.json`；發現 GitHub Pages 已發布新版資料時，顯示重新載入提示。
- 候選人公開頁移除司法狀態篩選與司法摘要，避免未經官方來源核實的資訊出現在公開介面。
- 候選人卡片改顯示正式來源數量。

## 安全與資料原則

- 不修改已同步的 `data/` 內容。
- 不變更每日 03:00 與每週日 04:20 排程。
- 不變更內政部地方公職解析器與中選會歷屆匯入器。
- 本機 `file://` 開啟時不執行遠端新版本檢查。

## 升級方式

使用小更新檔覆蓋 repository 最外層的：

- `index.html`
- `app.js`
- `styles.css`
- `CHANGELOG_V6.1.5.md`

不需覆蓋 `data/`，因此不會清除 GitHub Actions 已同步的現任與歷屆資料。
