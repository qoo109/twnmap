PUBLISH

官方資料發布流程

  1. GitHub Actions 從政府官方來源取得資料。
  2. 執行格式驗證、人物身分比對與大量刪除保護。
  3. 驗證通過後，自動提交更新檔至 repository。
  4. GitHub Pages 重新部署後，公開網站取得新版資料。
  5. 若來源失敗或異動異常，保留上一版並在後台顯示警示，不以人工內容補洞。
