# V6.2 — 官方資料唯讀版

## 移除

- 候選人 JSON／CSV 人工匯入。
- 人工發布檔產生器與候選人範本。
- 人工補充政見、經歷、來源、照片與司法裁判的流程。
- 公開頁司法狀態篩選及人物司法摘要。
- `config/manual-overrides.json` 與人工候選人覆寫邏輯。

## 新增與調整

- 新增 `config/official-mappings.json`，只做官方欄位與政黨名稱標準化。
- 管理後台改為官方同步、資料健康、異動審核與發布驗證。
- 政見／經歷僅接受中選會公報或政府官方人物頁直接提供的欄位。
- 頭貼來源限政府機關官方頁面。
- 同步失敗時不以人工內容填補。

## 排程

- 每天 03:00：現任公職與正式候選人。
- 每週日 03:20：中選會核心歷屆資料。
