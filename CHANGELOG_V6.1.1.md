# V6.1.1｜立即完整更新版

## 新增

- 管理後台新增醒目的「立即完整更新」按鈕。
- 一鍵觸發 `daily-sync.yml`，固定使用 `history_scope=core` 與 `history_years=all`。
- 同步內容包含現任公職、正式候選人、中選會核心歷屆、官方頭貼、異動報告、縣市／人物／歷屆分片與驗證。
- 顯示 GitHub Actions 排隊、執行、成功、失敗與逾時狀態。
- 公開首頁新增「管理員更新」入口。
- 新增 `DATA_LICENSE.md`，說明政府開放資料授權、顯名、速率與禁止行為。

## 安全

- Token 不寫入 repository。
- 大量刪除守門維持啟用。
- 官方來源失敗時保留上一版。
- 未正式公告的候選年度不顯示。
