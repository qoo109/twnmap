PUBLISH

發布流程

  1. 下載更新後的 election-data.json 與 candidates.js。
  2. 替換 repository 內 data/ 的同名檔案。
  3. 執行 npm run validate，再提交到 GitHub。
  4. GitHub Pages 重新部署後，網站即更新；下一次每日同步會更新官方現任名單，並保留人工覆核的候選人司法資料。
  5. 歷屆結果匯入後，可執行 npm run sync:history 重建人物軌跡、政黨席次與選區歷史索引。
