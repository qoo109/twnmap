# V6.1.8｜地方公職分批同步與備份版

- 依診斷檔確認內政部解析器已成功讀取所有地方公職類別。
- 自動同步拆成三批：03:00 核心公職、03:40 地方代表、04:20 村里長。
- 歷屆同步移至每週日 05:20，避免流程互撞。
- 首頁資料不再因單次完整流程超過兩小時而全部作廢。
- 小型公職頁面在第一個有效回應後停止多餘探測。
- 地方同步跳過已知較慢的資料集探測，直接使用已驗證的內政部頁面。
- `npm run sync` 不再重複執行頭貼、索引及分片工作。
- 村里長與地方代表不建立一人一檔，避免產生近萬個 Git 檔案。
- 每批同步都產生 14 天 GitHub Actions 備份與診斷 Artifact。
- 官方頭貼改由後續獨立流程處理，現任資料同步不再被照片拖慢。
