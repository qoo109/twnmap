# V6.1.7 — 地方公職解析器第二代

## 重點修正

- 重新設計內政部地方公職 HTML 解析，不再依賴固定的四行相鄰順序。
- 以「詳細資訊」卡片邊界辨識姓名、縣市、職稱與黨籍。
- 當官方頁面省略詳細資訊連結時，改用職稱列回退解析。
- 支援內政部頁面無換行、無空白的 inline HTML 結構。
- 首頁會自動測試 200、100、50、20、16 筆等頁面大小，選擇可解析筆數最多者。
- 完整模式最多可依官方總筆數抓取 600 頁，不再被舊的固定 maxPages 截斷。
- 分頁採最多 3 個並行請求並保留請求間隔、重試與大量刪除保護。
- 每次同步輸出 `data/reports/moi-parser-diagnostics.json`，記錄每一類公職的頁數、筆數與解析標記。
- GitHub Actions 會保留 7 天診斷檔 artifact，方便同步失敗時定位官方頁面格式問題。
- 增加三種 MOI 實際頁面結構 fixture 測試。

## 資料規則

- 只採內政部、立法院、總統府與中選會等官方來源。
- 任一角色低於安全門檻時，保留上一版資料，不發布空白或異常大量刪除結果。
- 公開網站名稱仍為「台灣選舉報報」，不顯示內部版本號。
