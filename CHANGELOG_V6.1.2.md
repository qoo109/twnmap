# V6.1.2｜排程防碰撞版

## 排程調整

- 每天台灣時間 03:00：更新現任公職、正式候選人與官方頭貼。
- 每週日台灣時間 04:20：更新中選會核心歷屆資料。
- GitHub cron 使用 UTC，因此歷屆排程為 `20 20 * * 6`（UTC 週六 20:20）。

## 同步防碰撞

每日與歷屆 workflow 共用同一個 `election-data-sync` concurrency group，並設定 `queue: max`。若兩個更新同時被觸發，只會執行一個；另一個會排隊等待，不會同時修改並推送資料。

## 保留功能

- 每日 03:00 自動同步。
- 管理後台手動同步。
- 中選會歷屆資料匯入。
- 官方頭貼本地化。
- 資料異動報告與大量刪除保護。
- GitHub Pages 自動發布。
